# Angular Products Console

UI para gestionar y probar la Products API (FastAPI).

## Requisitos
- Node.js 18+.

## Ejecutar
```bash
npm install
npm run start
```

Abre `http://localhost:4200`.

## Configuración
- Cambia `apiBaseUrl` en `src/environments/environment.ts` si el backend no está en `http://127.0.0.1:8000`.
